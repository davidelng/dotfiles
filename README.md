# Dotfiles

My personal settings for common Linux tools.

## Content

- Nvim (with a customized [Kickstart Nvim](https://github.com/nvim-lua/kickstart.nvim))
- Tmux
- Zsh
- Bash

## Requirements

- Git
- [Exa](https://github.com/ogham/exa)
- [Neovim](https://github.com/neovim/neovim) >= 0.9.5
- A [NerdFont](https://www.nerdfonts.com/) to display icons
- [Tmux](https://github.com/tmux/tmux/wiki)
- [Flexoki colorscheme](https://github.com/kepano/flexoki) if needed

If using Zsh
- [Zsh Syntax Highlighting](https://github.com/zsh-users/zsh-syntax-highlighting/blob/master/INSTALL.md)
- [Zsh Autosuggestions](https://github.com/zsh-users/zsh-autosuggestions)
